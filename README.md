# Python Data Analysis Practice

Python data analysis practice using pandas and matplotlib. Includes:

- CSV loading
- Filtering
- Aggregation
- Basic visualization

## How to Run

1. Make sure you have Python installed (preferably 3.x)  
2. Install required libraries:
   ```bash
   pip install pandas matplotlib
   ```
3. Place people.csv in the data/ folder
4. Run the script:
   ```bash
    python people.py
   ```

## Skills Demonstrated
- Python
- Pandas
- Matplotlib
- Data filtering and aggregation
- Basic data visualization
